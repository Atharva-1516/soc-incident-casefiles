SOC Incident Casefiles

This repository contains structured incident investigations simulating real-world SOC triage scenarios.

Each casefile includes:

Alert summary

Initial hypothesis

Log analysis

Indicators of Compromise (IOCs)

Detection logic

MITRE ATT&CK mapping

False positive considerations

Remediation steps

Lessons learned

The goal of this project is to demonstrate:

Analytical thinking

Log correlation skills

Detection engineering mindset

Structured investigation workflow

Clear documentation skills
